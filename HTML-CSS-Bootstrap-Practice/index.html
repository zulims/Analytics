Summary: Latitude vs. X

This project analyzes the relationship between weather and latitude. It attempts to answer questions like “does data show temperature increase as you get closer to the equator?” and “is there a correlation between proximity to the equator and cloudiness?”

Using the Open Weather API and the CitiPy Python Library we generated 500 random coordinate combinations to create a list of unique cities all over the globe along with their weather data. We then used Python Pandas and Matplotlib to analyze this data. The four variables we looked at were: max temperature, humidity, cloudiness, and wind speed. This dashboard allows you to see our results side by side, and dive deeper to get a longer description of each plot.
